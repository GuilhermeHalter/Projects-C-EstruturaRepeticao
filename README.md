# Atividades em C sobre Estrutura de Repetição.

**Estrutura do "for"**

```C
for ([variavel inicial];[critério];[contador]){
    /*
    codificação (verdadeira);
    */
}
```

**Estrutura do "while"**

```C
while ([critério]){
    // codificação
}
```

**Estrutura do "do while"**

```C
do {
    // codificação
} while ([critério])
```
